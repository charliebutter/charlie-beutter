Fantasy Name Generator

A powerful tool for creating fantasy-style names with customizable themes and linguistic features that's perfect for writers and worldbuilders seeking names that fit specific fantasy cultures. The generator works by intelligently combining letter "blocks" based on configurable parameters. Using a sophisticated scoring system, it selects blocks that work together to match your desired characteristics.

[bitmask]

[bitmask] is a unique grid-based logic puzzle that combines parallel constraints and deductive reasoning into an accessible yet challenging gameplay experience similar to Sudoku or Nonograms. Finding the grid that satisfies all constraints requires careful reasoning about the interplay between multiple overlapping conditions, making each solution discovery genuinely satisfying.

mise

Mise is a recipe management application that helps users organize their culinary collection. It features user authentication, recipe creation and editing, tagging and note-taking capabilities, and favorites management. The app provides a clean, intuitive interface for storing and accessing personal recipes.

Shrouded

Shrouded is a puzzle game where players arrange 8 letters to find hidden words. It features intelligent puzzle generation using backtracking algorithms, shareable puzzles via URL encoding, and progressive hints. Players must discover valid letter arrangements while managing multiple overlapping words.
