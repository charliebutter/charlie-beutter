Fantasy Name Generator

A powerful tool for creating fantasy-style names with customizable themes and linguistic features that's perfect for writers and worldbuilders seeking names that fit specific fantasy cultures. The generator works by intelligently combining letter "blocks" based on configurable parameters. Using a sophisticated scoring system, it selects blocks that work together to match your desired characteristics.

mise

Mise is a recipe management application designed to be as simple and clean as possible. It features basic user authentication, recipe creation and sharing, tagging and favoriting, and collections management. The app provides an intuitive interface for storing and accessing personal recipes.

NovelForm

NovelForm is a minimal text viewer and editor designed primarily for reading novels from markdown files. It supports standard markdown formatting — bold, italics, and headers — as well as novel-specific formatting like paragraph indentation and scene dividers. It offers multiple color themes and fonts, paragraph bookmarking to save your place, and can import and export markdown files.

- web games -

Shrouded

Shrouded is a word game where players arrange 8 letters to find 4 hidden words. It features intelligent puzzle generation, shareable puzzles, and progressive hints. Players must discover a letter arrangement that overlaps with the highlighted tiles to uncover all four shrouded words.

[bitmask]

[bitmask] is a unique and challenging grid-based logic puzzle that combines parallel constraints and deductive reasoning into a difficult gameplay experience similar to Sudoku or Nonograms. Finding the grid that satisfies all constraints requires careful reasoning about the interplay between multiple overlapping conditions, making each solution discovery feel truly satisfying.

- external projects -

Shadowed

Shadowed is a custom room for Escape Simulator 2, created for the game's first Build-a-Room competition where it placed in the top 10 out of 112 entries. Every puzzle is built around a single theme: light, shadow, and perspective. Rather than relying on animations or screens, all puzzles are made practically using the real-time lighting engine. The result is a cohesive, atmospheric experience where paying attention to what you can and can't see is the key to everything.
